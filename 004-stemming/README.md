# Stemming Telugu Text

## Using nltk and indic-nlp-library

```text
nltk==3.8.1
```

### Steps to stem Telugu text

```bash
$ python3 -m venv .venv  # create a virtual environment if not already created
$ source .venv/bin/activate
(.venv) $ pip install --upgrade pip  # make sure pip is up-to-date
(.venv) $ pip install -r requirements.txt
(.venv) $ python stemming_using_porter_stemmer.py 
(.venv) $ deactivate
$ 
```
